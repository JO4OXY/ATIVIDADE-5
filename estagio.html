Relatório de Estágio

1. Introdução

O relatório de estágio supervisionado no curso Técnico em Informática visa documentar as atividades realizadas e os conhecimentos adquiridos durante o período de prática profissional.

1.1. Objetivos

1.1.1. Objetivo Geral

Aplicar os conhecimentos teóricos adquiridos durante o curso técnico em um ambiente profissional, contribuindo para o desenvolvimento de habilidades práticas.

1.1.2. Objetivos Específicos

Desenvolver um sistema de gerenciamento de tarefas, aprimorar habilidades de programação e trabalhar em equipe.

2. Atividades realizadas

2.1. Desenvolvimento do Sistema de Gerenciamento de Tarefas

Participação no desenvolvimento de um sistema de gerenciamento de tarefas utilizando a linguagem de programação Python e o framework Django.

2.2. Testes e Implementação de Funcionalidades

Realização de testes unitários e de integração, além da implementação de novas funcionalidades no sistema.

3. Conclusão

O estágio proporcionou uma experiência valiosa, permitindo a aplicação prática dos conhecimentos adquiridos no curso técnico e contribuindo para o desenvolvimento profissional.
